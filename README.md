# 📚 BookTrack — Sistema de Biblioteca Pessoal
 
Sistema simples em **Python** para gerenciar usuários e livros de uma biblioteca pessoal. Projeto desenvolvido para a disciplina de **Algoritmos e Programação** (UniSENAI Digital), com foco em funções, procedimentos, vetores e matrizes.
 
## 🧩 Sobre o projeto
 
A BookTrack é uma startup fictícia que precisa de um sistema para ajudar leitores a organizarem seus livros, já que hoje tudo é feito em planilhas soltas. Este repositório contém a versão inicial desse sistema, rodando no terminal.
 
O sistema permite:
 
- **Usuários:** cadastro, edição, exclusão, login e listagem
- **Livros:** cadastro, edição, remoção, listagem e acompanhamento de progresso de leitura
- **Relatório geral:** uma tabela (matriz) com o resumo de todos os livros cadastrados
## 🗂️ Estrutura do projeto
 
```
biblioteca/
├── main.py         # Menu principal e fluxo do programa
├── usuarios.py     # Cadastro, edição, exclusão, login e listagem de usuários
├── livros.py       # Cadastro, edição, remoção e listagem de livros
└── relatorios.py   # Geração da matriz-relatório de livros
```
